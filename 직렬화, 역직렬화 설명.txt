직렬화, 역직렬화 설명

✅ 직렬화 (Serialization)
정의: 객체나 데이터 구조를 **파일, 메모리, 네트워크 등으로 전송 가능한 형식(예: JSON, XML, 바이너리 등)**으로 변환하는 과정
목적: 객체를 저장하거나 전송할 수 있게 하기 위함
예시:
const obj = { name: "홍길동", age: 25 };
const json = JSON.stringify(obj);  // 직렬화
console.log(json);  // '{"name":"홍길동","age":25}'

✅ 역직렬화 (Deserialization)
정의: 직렬화된 데이터를 다시 원래의 객체 형태로 복원하는 과정
목적: 전송받거나 읽어온 데이터를 실제 프로그램에서 사용할 수 있는 형태로 변환하기 위함
예시:
const str = '{"name":"홍길동","age":25}';
const obj = JSON.parse(str);  // 역직렬화
console.log(obj.name);  // '홍길동'

✅ 언제 사용하나요?
네트워크 통신 시 (예: 클라이언트 ↔ 서버)
파일 저장 시 (예: 설정 파일)
데이터베이스나 캐시 시스템에서 저장/복원할 때

✅ 한 줄 요약
직렬화: 객체 → 전송 가능한 데이터 (JSON 등)
역직렬화: 데이터 → 다시 객체


✅ 정리하면
직렬화		데이터를 전송/저장을 위해 문자열 등으로 변환하는 과정
JSON			그 변환 결과를 담는 형식(포맷) 중 하나
JSON.stringify	자바스크립트 객체를 JSON 문자열로 직렬화
JSON.parse		JSON 문자열을 객체로 역직렬화
