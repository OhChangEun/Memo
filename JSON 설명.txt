JSON은 JavaScript Object Notation의 약자로, 
데이터를 저장하거나 전송할 때 사용하는 가볍고 사람이 읽기 쉬운 텍스트 형식의 데이터 포맷입니다. 
주로 웹 애플리케이션에서 서버와 클라이언트 간 데이터 교환에 사용됩니다.

✅ JSON의 특징
텍스트 기반이라 사람도 쉽게 읽고 쓸 수 있음
대부분의 프로그래밍 언어에서 쉽게 사용할 수 있도록 지원됨
키-값 쌍으로 구성된 구조 (파이썬의 딕셔너리, 자바스크립트의 객체와 유사)

✅ JSON 구조 예시
{
  "name": "홍길동",
  "age": 25,
  "isStudent": true,
  "skills": ["C#", "Python", "JavaScript"]
}

"name"은 키(key), "홍길동"은 값(value)

값은 문자열, 숫자, 불리언, 배열, 또 다른 JSON 객체 등이 될 수 있음

✅ 사용 예
웹에서 서버로 데이터를 보낼 때:
{ "id": 101, "command": "start" }
API 응답 결과로 JSON을 주는 경우가 많음 (예: 날씨 정보, 사용자 정보 등)
